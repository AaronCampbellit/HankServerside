Hank Remote

Setup Guide

Run Hank Remote on one server and point the Hank app at it.

Live Server

Use a reverse proxy or Cloudflare Tunnel for the public address.

Simple Picture

Hank app -> Hank Remote server -> Home connector

The app talks to the server. The home connector keeps your local Home Assistant, files, and notes reachable.

Server Settings

.env.cloud
Private server file for database and backup settings.
Address
Default server address is port 18080.
Database
Postgres stays private inside Docker.
Backups
Encrypted backup settings live in .env.cloud.
Login Lifetime
How long app sessions stay signed in.
Request Timeout
How long the server waits for the home connector.

Home Connector Settings

Server Link
The connector uses this to reach Hank Remote.
Connector ID
The name used to recognize this connector.
Setup Token
The private token created by the dashboard.
Home Name
The friendly name shown in the dashboard.

First Checks

1. Server running returns OK.
2. Database ready returns OK.
3. First account becomes admin.
4. Later registration is blocked.
5. The home connector shows online.
6. Storage backup and restore test pass.
7. The Hank app can log in.
8. Files, notes, or Home Assistant load from the app.
