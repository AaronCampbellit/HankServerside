Agent Runtime

Single Host

The cloud and agent always run together on this same machine under one Docker Compose stack. Use the dashboard to issue tokens, then update .env.agent and restart the agent service.

Runtime Shape

The public cloud service listens on 127.0.0.1:8080 and is exposed through Cloudflare Tunnel. The local agent connects to ws://cloud:8080/ws/agent over the internal Docker network.

Agent Config

Fill in Home Assistant and SMB values in .env.agent. Leave the SMB settings blank to keep using the mounted local files directory at /srv/hank/files.

Apply Changes

After issuing a new token or editing .env.agent, restart only the agent container with docker compose up -d --no-deps agent.
